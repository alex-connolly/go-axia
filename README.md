# go-axia

Golang implementation of the Axia protocol.

## Overview

Axia is a consensus protocol which allows for the storage of arbitrary state on the blockchain, and the deterministic execution of Turing complete smart contracts. The protocol uses a new cryptocurrency, also called Axia (denoted Ó¾), as the primary store of value.

## Protocol

The Axia protocol is composed of three sub-protocols:

- Network
- Validation
- Consensus
